Find and Analyse Reviews

Fast and convenient way to find out what is written about your product or service in Twitter and anylise tweet text

About Project

This project allow you to make fast and easy reviews search about your product or service via Twitter. It makes a research for keyword in Twitter (using Twitter API) and store received information into array. Next, using Microsoft Cognitive Service API it analyses tweets texts. Finally, all the information is shown on the page in convinience format.

Technologies used

Web application was developed with usage of:

  • HTML 5 and CSS 3
  • Bootstrap for responsiveness
  • TypeScript for the logic, dynamic API calls
  • jQuery - AJAX calls and rendering results
  • SCSS for easy CSS managing
  • Gulp for syncronising and compiling
  • GitHub for Source Control
  • Microsoft Azure for hosting and publishing
  • Continuous Deployment via GitHub
  • Application Insights for monitoring
  • Facebook Sharing just for fun
  • Twitter API
  • Microsoft Cognitive Services API (text analysis)
  • NodeJS for back-end
  • JQuery FakeLoader plugin for loading layer
  • JQuery SmoothSlider plugin for smooth page scroll
